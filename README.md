# Makhokolotso Maria Khapetsi Portfolio Website

## Overview

This is a modern, responsive personal portfolio website built for **Makhokolotso Maria Khapetsi** — an Author, Entrepreneur, Mentor, Speaker, and Women's Empowerment Advocate.

The website showcases her personal journey, published books, empowerment initiatives, mission, and contact information while providing visitors with an elegant and engaging user experience.

## Features

### Home Page

* Full-screen hero section
* Inspirational introduction
* Professional branding
* Smooth animations and transitions
* Call-to-action button

### About Page

* Detailed biography
* Typing text animation
* Scroll-triggered fade effects
* Professional portrait section
* Expandable "Read More" content

### Books Page

* Showcase of published works
* Product images
* Pricing information
* WhatsApp purchase integration
* Expandable book descriptions

Books Featured:

* My Prayer Journal
* Her Story, Her Journey

### Ventures Page

Highlights Makhokolotso's organizations and businesses:

* Single Moms with Purpose
* Hey Queen Clothing Brand

### Mission Page

Displays:

* Vision statement
* Core values
* Empowerment message
* Faith-centered purpose

### Contact Page

Interactive contact form powered by EmailJS.

Users can:

* Send inquiries
* Request speaking engagements
* Discuss collaborations
* Submit messages directly from the website
